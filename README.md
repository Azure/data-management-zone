# Description 
The Data Management Zone is, as the name suggests, classified as a management function and is at the heart of the Enterprise Scale Analytics platform. It is responsible for the data governance of the platform and ensures proper handling of datasets within and outside of the Azure environment.

## What will be deployed?

By default, all the services which comes under Data Landing Zone are enabled and you must explicitly disable them if you don't want it to be deployed.
